Lumen 5.2 正式发布

作者： 王占康 2016年1月27日

Lumen 5.2.0 已经正式发布了！此次发布的版本包含了 Laravel 5.2 的组件并在核心理念上做了重大改变。 建议所有 Lumen 用户阅读相关文档了解以下所提到的改变： 专注提供无状态的 API 服务 Lumen 5.2 对 Lumen 进行了大瘦身专注于，并且将专注提供无状态的 JSON API 服务。
